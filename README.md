# Wipehouse ecomerce
Proyecto personal para aprender a utilizar React. 

El objetivo es realizar una web ecomerce mediante React utilizando distintas funcionalidades de Firebase. 
Usando Redux para gestionar los estados apoyandonome en Redux-Saga para manejar los side effects.

## Descripción

Wipehouse_ecomerce es una web que permite la compra de servicios domésticos. Mediante un buscador puedes añadir distintos servicios al carrito para más adelante realizar la compra. 

Los servicios se filtran según su categoría actualmente están disponibles:
* Cocina a domicilio
* Mantenimiento de piscinas
* Limpieza del hogar
* Cortacésped

Cuenta con un sistema de registro de usuarios en donde se puede consultar los pedidos realizados y sus detalles. Además si el usuario cuenta con permisos de administrador tendrá acceso a un panel de administrador donde podrá añadir o eliminar servicios.
